BGObjectRoutingBundle
===================

This bundle allows you to use the [object-routing](http://jmsyst.com/libs/object-routing) library.

Installation
------------

Instal the bundle with composer:

```json
    "require": {
``` 

```bash
$ php composer.phar update benjam1/object-routing-bundle
```
